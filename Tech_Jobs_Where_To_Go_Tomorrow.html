🎯 Walk-in / interview playbook — for a CSE fresher

  • Call before you go. Ring HR (numbers above), say "B.Tech CSE 2026, I applied online — can I come for a walk-in interview / drop my resume?" Get a name + time. Don't show up cold at a secured office.
  • Carry 4–5 printed resumes + a one-line GitHub / project link at the top. For these product companies, a working project beats marks.
  • Know one project + DSA basics cold. Be ready to explain a project end-to-end and solve an easy/medium array/string problem. For AI roles (AgNext, XenonStack, Nerve Cells), revise Python + one ML project.
  • Best time to visit: ~10:30 AM–12:00 or 3:00–5:00 PM, Mon–Thu. Dress smart-casual. The whole cluster is at one address, so you can line up 2–3 in a day.
  • Follow up same day by email/LinkedIn to the HR you spoke to. Connect with the founder where listed (e.g. Nerve Cells AI).
